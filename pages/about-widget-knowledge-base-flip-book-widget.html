What this widget does

Allows ability to insert images in a book styled gallery. Great for e-zines or publishing agencies.

Common uses

  • Add a focused feature to a Duda website without rebuilding the whole page.
  • Improve how visitors interact with calls to action, media, content, or business information.
  • Keep widget setup organized so the feature can be adjusted later by WebAct or the website owner.

Setup notes

Widget options vary by feature, but most WebAct widgets include content settings, design controls, display options, and support paths for matching the widget to the website's brand and layout.
