FAQ

Microsoft Advertising questions businesses ask before starting.

Is Microsoft Advertising the same as Bing Ads?

Bing Ads is now Microsoft Advertising. The platform includes Bing search ads along with Microsoft Audience Network, shopping, remarketing, and additional Microsoft placements.

Should I run Microsoft Ads if I already use Google Ads?

Often, yes. Microsoft Ads can add incremental reach and may help lower dependency on one platform.

Can WebAct import Google Ads campaigns into Microsoft Ads?

Microsoft allows Google Ads imports, but campaigns should still be reviewed and adjusted for Microsoft traffic, settings, budgets, and performance differences.

Do Microsoft Ads need landing pages?

Yes. Strong landing pages help improve conversion rates, message match, trust, and lead quality after someone clicks an ad.
